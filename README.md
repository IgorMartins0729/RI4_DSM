# Sistema de Empresa e Clientes (JavaScript)

## 📌 Descrição

Este projeto implementa um sistema simples de cadastro de **empresa e clientes** utilizando **Programação Orientada a Objetos em JavaScript**.

O sistema permite:

* cadastrar clientes
* cadastrar telefones
* cadastrar empresa
* associar clientes à empresa
* gerar um relatório com os dados da empresa e dos clientes

---

# 🧱 Estrutura do Projeto

```
app.js
models.js
README.md
```

* **models.js** → contém as classes do sistema
* **app.js** → arquivo principal que executa o programa

---

# 🚀 Tecnologias
